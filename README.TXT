Comandos GIT
git init -> inicia o controle de versão dentro de um diretório repositório de códigos

git clone -> clonar o projeto de um repositório remoto para o local

git add . ou gti add <<nome do arquivo>> -> adiciona todos os arquivos do diretório ou um arquivo

git commit -m 'mensagem de commit' -> salvar as alterações feitas no arquivo já adicionado

git push -> publica as alterações num servidor remoto

git pull -> baixar as alterações do servidor remoto para meu repositório local

git merge -> juntar as alterações de uma determinada branch com sua branch principal

git status -> provê informações referentes ao estado atual do projeto.

git log -> devolve o histórico de alterações do projeto

git branch <<nome da branch>> -> cria um novo ramo (master que é o de produção, e outros ramos de acordo com suas necessidades de implementação)

visite o site: https://git-scm.com/docs
